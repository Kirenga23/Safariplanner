🔑 How to activate the AI — One-time Vercel setup
Step 1: Go to console.anthropic.com → Sign up free → API Keys → Create Key → copy it
Step 2: Go to vercel.com → Your safariplanner project → Settings → Environment Variables
Step 3: Add variable: Name = ANTHROPIC_API_KEY · Value = your key (starts with sk-ant...)
Step 4: Click Save → Go to Deployments → click Redeploy
Done! The AI will work on your next visit · Imekamilika! AI itafanya kazi ziara yako inayofuata
